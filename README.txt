Consideraciones:
	- Para los nombres de servidores duplicados, se mantiene en el diccionario el último servidor ingresado.
	- También que un nombre de servidor no puede tener espacios en blanco entre medio. Por ejemplo: "servidor 1"
	- El cliente esta escrito en Python 3, por lo que el comando para ejecutarlo es: 'python3 client.py'
	- Se diferencian las respuestas de los servidores usando el nombre asignado.
	- Se incluye un archivo .json de ejemplo.
	- El usuario tiene un tiempo de 10 segundos para ingresar el comando o nombre de los servidores a enviar el comando.
	- En caso de que no alcanze a escribir el comando, aunque no se muestre este queda en cache y el usuario solo debe continuar de donde quedo.

	
Comandos:

	- Correr cliente:
		python3 client.py
	
	- Terminar cliente:
		control + c [Interrupcion de teclado]
